Как использовать фильтрацию
  1. Установите минимальный процент совпадения - регулируйте ползунок для изменения порога совместимости
  2. Выберите конкретную технологию - фильтруйте тендеры по наличию определенной технологии
  3. Используйте даты анализа - ищите тендеры, проанализированные в определенный период
  4. Просматривайте статистику - анализируйте частоту упоминания технологий в тендерах
  5. Переходите к документам - для детального изучения нажмите на иконку документа
Совет: Для более точной фильтрации сначала скачайте и проанализируйте извещения (ТЗ) тендеров на странице документов.
