Критерии фильтрации

Настройте фильтры для уведомлений о новых тендерах. Уведомления будут отправляться только для тендеров, соответствующих критериям.

Ключ Значение Действие
{{ key }} {{ value }}
Примеры критериев:
• minPrice: 100000 (минимальная цена)
• maxPrice: 500000 (максимальная цена)
• region: "Москва" (регион)
• customerInn: "1234567890" (ИНН заказчика)
• purchaseType: "строительство" (тип закупки)
Сообщение для тестового уведомления
Информация о системе уведомлений
  • Система проверяет новые тендеры каждые 5 минут
  • Уведомления отправляются только если включены соответствующие опции
  • Для работы email уведомлений настройте SMTP в конфигурации
  • Для работы Telegram уведомлений настройте бота и укажите Chat ID
  • Webhook уведомления отправляются в формате JSON
